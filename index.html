  1. The Three main diffrences between web1,web2 and web3:

    1. The main diffrence between the three, is that web 1.0 is read-only, web 2.0 is writable and allows interaction between web users and sites,
      and web 3.0 is a semantic web in which computers can understand information like humans and distribute custom content.
    2. In web 1.0 the content is made by big companies, in web 2.0 the content is made people through platforms made by big companies,
      in web 3.0 both content and different services are made by individuals, and big big comapanies mainly serve as a platformn
      in which those services can be presented.
    3. In web 1.0 query results will be very general and will include extra content which probably will not be relevant,
      in web 2.0, everything is taged manualy, so results probably will be more relevant and not include as mush extra content as in web 1.0,
      in web 3.0 everything automaticly, so results will be taken from a much bigger database and will be a lot more directed to the specific query.

  2. Medicines side effects Use Case:

    1. UC in web 1.0:
      1. The user enters the query "Medicines side effects" into search engine.
      2. The user clicks enter.
      3. The user is presented with various official drug companies sites.
      4. The user clicks on the ooficial website of the company than made the medicine.
      5. The user is presented with generic information about the drug, which is probably allready written in the instructions.

    2. UC in web 3.0:
      1. The user enters the query "Medicines side effects" into search engine.
      2. The user clicks enter.
      3. The user is presented with various diverse site which may include personal experiences with the drug,
        statements by unbiased medical proffesionals, and websites that offer aoutomatic solutions .
      4. The user clicks on a site which offers him personalized assistance.
      5. The user is presented with a survey, in which he fills his personal details.
      6. The user is then presented with answers about the medicine that are tailored for his personal condition.li>
